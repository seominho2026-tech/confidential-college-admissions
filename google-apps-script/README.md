# 🎓 구글 앱스 스크립트(Google Apps Script) 웹앱 적용 가이드

이 가이드는 작성된 `Code.gs`와 `Index.html` 코드를 당신의 구글 스프레드시트에 직접 설치하고 독립적인 **대입 수시 결과 대시보드 웹앱**으로 배포하는 단계를 상세히 알기 쉽게 설명합니다.

---

## 📋 사전 준비사항
1. 사용하고자 하는 구글 스프레드시트를 엽니다.
   - 단, 시트의 **이름은 반드시 `수시`**로 설정되어 있어야 합니다.
   - 시트의 컬럼 순서 및 구성은 아래와 일치해야 합니다:
     - **A열**: 학년도 / **B열**: 지역 / **C열**: 대학명 / **D열**: 전형유형 / **E열**: 세부유형 / **F열**: 계열 / **G열**: 학과 / **H열**: 환산등급 / **I열**: 환산점수 / **J열**: 합불여부 등...

---

## 🚀 6단계 웹앱 설치 및 배포 절차

### 1단계: 구글 앱스 스크립트 실행
1. 구글 스프레드시트 상단 메뉴에서 **[확장 프로그램]** ➔ **[Apps Script]**를 클릭합니다.
2. 잠시 후 구글 앱스 스크립트 에디터 화면(새 브라우저 탭)이 열립니다.

### 2단계: 서버측 스크립트(`Code.gs`) 입력
1. 기존에 기본값으로 적혀 있던 `function myFunction() { ... }` 코드를 **전부 삭제**합니다.
2. 이 디렉토리에 생성되어 있는 `Code.gs` 파일의 전체 코드를 복사해서 붙여넣습니다.
3. *참고:* `Code.gs`의 14행 근처에 `SPREADSHEET_ID = "17BPrRpIKTw8LRBnT7vPQgTOs7Nk1cy7278zn_5c2_gQ"`가 이미 들어가 있으므로 별도로 아이디를 조회하지 않아도 기본 탑재된 상태입니다. 필요 시 다른 시트 ID로 바꿀 수 있습니다.

### 3단계: 클라이언트측 UI 파일(`Index.html`) 생성 및 입력
1. 앱스 스크립트 화면의 왼쪽 파일 목록에서 **[+] (파일 추가)** 버튼을 누른 뒤 **[HTML]**을 선택합니다.
2. 새 파일명으로 `Index`를 입력합니다. (확장자 `.html`은 자동 생성되므로 **`Index`만 입력**해야 합니다)
3. 생성된 `Index.html` 파일의 내용을 전부 지우고, 이 디렉토리에 들어있는 `Index.html` 파일의 코드를 복사하여 그대로 붙여넣습니다.

### 4단계: 소스코드 저장
- 코드 에디터 상단의 💾 **[프로젝트 저장]** 아이콘을 클릭하거나 단축키 `Ctrl + S` (macOS는 `Cmd + S`)를 눌러 수정한 내용을 모두 저장합니다.

### 5단계: 웹앱으로 배포하기 (Deployment)
1. 오른쪽 상단의 파란색 **[배포]** 버튼을 누르고 **[새 배포]**를 선택합니다.
2. 왼쪽 톱니바퀴 아이콘 **[유형 선택]**에서 **[웹앱]**을 클릭합니다.
3. 설정 옵션들을 다음과 같이 맞춥니다:
   - **설명**: `대입 수시 결과 대시보드 배포` (원하는 임의 문구)
   - **웹앱을 실행할 사용자**: `나 (seominho2026@gmail.com 등 귀하의 이메일)`
   - **액세스 권한이 있는 사용자**: `모든 사람` *(이 항목을 '모든 사람'으로 설정해야 인증서 바벨 오류 없이 정상 기동됩니다)*
4. 하단의 **[배포]** 버튼을 클릭합니다.

### 6단계: 구글 계정 액세스 승인 (최초 1회 필수)
- "액세스 승인(Authorization)" 창이 나타나면 아래 절차를 진행합니다.
- **[액세스 승인]** 버튼 클릭 ➔ 본인의 **구글 계정 선택**합니다.
- 경고 페이지가 열리면 당황하지 마시고 좌측 하단의 **[Advanced] (또는 고급)** 링크를 클릭한 뒤, **[Go to 대입 수시 결과 대시보드 (unsafe) / 이동]**을 클릭합니다.
- 마지막으로 수시 데이터 로드를 위해 **[Allow] (또는 허용)**을 클릭합니다.

## 🎉 배포 완료 및 접속
- 승인이 완료되면 **"웹앱 URL"**이 생성되어 화면에 출력됩니다!
- 해당 URL을 복사하여 즐겨찾기 하거나 브라우저 주소창에 붙여넣으면, 모바일 및 데스크톱 어디서든 실시간으로 대학 수시 데이터를 쾌적하게 분석할 수 있는 **프리미엄 반응형 대시보드**가 열립니다.
